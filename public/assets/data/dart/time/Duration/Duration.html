/** 一、概念： 1、Duration表示从一个时间点到另一个时间点的时间差，即：多少天 ｜ 多少小时｜多少分钟｜多少秒｜多少毫秒｜多少微秒 2、如果是一个较晚的时间点和一个较早的时间点，Duration可能是负数 */ # 一起使用 Duration duration = Duration({ int days: 1, // 天数 int hours: 1, // 小时数 int minutes: 1, // 分钟数 int seconds: 1, // 秒数 int milliseconds: 1, // 毫秒数 int microseconds: 1, // 微秒数 }) // => 25:01:01.001001 # 单独使用 Duration duration = Duration(days: 1); // => 24:00:00.000000 Duration duration = Duration(hours: 1); // => 1:00:00.000000 Duration duration = Duration(minutes: 1); // => 0:01:00.000000 Duration duration = Duration(seconds: 1); // => 0:00:01.000000 Duration duration = Duration(milliseconds: 1); // => 0:00:00.001000 Duration duration = Duration(microseconds: 1); // => 0:00:00.000001
