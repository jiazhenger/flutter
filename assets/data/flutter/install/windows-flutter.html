# 安装 flutter sdk // 方法一 到flutter 官网下载 // 方法二 git clone -b stable https://github.com/flutter/flutter.git # 配置环境变量， 在用户变量的path中配置 D:\flutter-sdk\flutter\bin # 配置国内境像，在系统变量配置 FLUTTER_STORAGE_BASE_URL ======= https://storage.flutter-io.cn PUB_HOSTED_URL ======= https://pub.flutter-io.cn # npm 命令检测安装依赖，如果没有，则安装；反之，则显示依赖信息 > flutter doctor # 运行到移动设备 > flutter run # 开启 flutter web 支持 > flutter config --enable-web # 为已存在的项目添加 flutter web 支持 > flutter create . # 运行到浏览器 > flutter run -d chrome # 打包成 fluuter web > flutter build web
