# contains(value)：列表中是否包含指定元素 bool v = list.contains(3); # elementAt(index)：返回列表中的第几个元素 var v = list.elementAt(3); # indexOf(value)：返回列表中第一个匹配元素的下标，从前向后匹配 int v = list.indexOf(3); // 返回 value=3 的下标 # lastIndexOf(value)：返回列表中最后一个匹配元素的下标，从后向前匹配 int v = list.lastIndexOf(3); // 返回 value=3 的下标 # sublist(startIndex,endIndex)：返回从开始下标（包括）到结束下标（不包括）元素的列表 int v = list.sublist(1,3); // 返回第1~3下标的值组成的列表 int v = list.sublist(1); // 返回从第1个下标（包括）到最后一个元素的列表
