# 安装 flutter sdk // 方法一 到flutter 官网下载 // 方法二 git clone -b stable https://github.com/flutter/flutter.git # 配置环境变量， 在用户变量的path中配置 D:\flutter-sdk\flutter\bin # 配置国内境像，在系统变量配置 FLUTTER_STORAGE_BASE_URL ======= https://storage.flutter-io.cn PUB_HOSTED_URL ======= https://pub.flutter-io.cn # npm 命令检测安装依赖，如果没有，则安装；反之，则显示依赖信息 > flutter doctor
