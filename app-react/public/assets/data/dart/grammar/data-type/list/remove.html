# remove(value)：移除列表中匹配的第一个元素 list.remove(3); // 移除列表中第一个 value = 3 的元素 # removeAt(index)：移除列表中第几个元素 list.removeAt(3); // 移除列表中第 3 个的元素 # 移除列表中最后一个元素 list.removeLast( ); # 移除列表的所有元素 list.clear( ); # removeRange(startIndex,endIndex)：移除开始下标（不包括）至结束下标（包括）内的元素 list.removeRange(0,4); // 移除索引值 0~4 元素， # 移除所有满足条件的元素，此为Dart的一种速写语法 list.removeWhere((value) => value == 3); // =3 的移除， # 移除所有不满足条件的元素，此为Dart的一种速写语法 list.retainWhere((value) => value == 3); // !=3 的移除，只留 3
